pycleora.embed()

embed(graph, feature_dim=128, num_iterations=4, propagation="left", normalization="l2", seed=0, initial_embeddings=None, num_workers=None, callback=None, residual_weight=0.0, convergence_threshold=0.0, whiten=False)

Main embedding function. Automatically uses fast Rust path when possible (no callback, L2 norm, no custom init).

graph : SparseMatrix

feature_dim : int — Embedding dimensionality (default: 128)

num_iterations : int — Propagation iterations (default: 4)

propagation : str — "left" or "symmetric"

normalization : str — "l2", "l1", "spectral", "none"

seed : int — Deterministic initialization seed

initial_embeddings : np.ndarray | None — Custom initial vectors

num_workers : int | None — Thread count

callback : Callable[[int, np.ndarray], None] | None — Per-iteration callback

residual_weight : float — Mix with previous iteration (0 = off). v3.2

convergence_threshold : float — Early stopping RMSE threshold (0 = off). v3.2

whiten : bool — Apply whitening post-processing. v3.2

Returns: np.ndarray — Shape (n_entities, feature_dim)

pycleora.algorithms

embed_deepwalk(graph, feature_dim=128, num_walks=10, walk_length=80, window_size=5, seed=0)

DeepWalk: uniform random walks + SVD of co-occurrence matrix.

embed_node2vec(graph, feature_dim=128, num_walks=10, walk_length=80, window_size=5, p=1.0, q=1.0, seed=0)

Node2Vec: biased random walks. p = return (BFS), q = in-out (DFS).

embed_netmf(graph, feature_dim=128, window_size=5, negative_samples=1.0)

NetMF: closed-form matrix factorization equivalent to DeepWalk.

embed_prone(graph, feature_dim=128, mu=0.2, theta=0.5, seed=0)

ProNE: spectral propagation with band-pass filtering.

embed_randne(graph, feature_dim=128, num_iterations=3, weights=None, seed=0)

RandNE: Gaussian random projection + power iteration. Ultra-fast.

embed_grarep(graph, feature_dim=128, max_step=4)

GraRep: multi-scale transition matrix factorization.

embed_hope(graph, feature_dim=128, beta=0.1)

HOPE: Higher-Order Proximity via Katz similarity.

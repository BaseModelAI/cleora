Changelog

All notable changes to pycleora, organized by version.

v3.0.0

March 2026 — Major Release

New Mini-GNN (GCN) Classifier

Pure numpy/scipy 2-layer Graph Convolutional Network. No PyTorch or TensorFlow required. Symmetric normalization (D^{-1/2} A_hat D^{-1/2}), dropout, multi-layer support. 100% accuracy on Karate Club dataset.

New Graph Sampling Module

6 sampling methods: neighborhood sampling (k-hop), subgraph sampling, GraphSAINT mini-batching, negative sampling, train/test edge splits, and BFS-based sampling. Enables scalable training on large graphs.

New Negative Sampling

Efficient negative edge generation for link prediction. Configurable negative-to-positive ratio. Integrates with train/test splits.

New Cross-Validation

k-fold cross-validation for node classification. Reports per-fold accuracy and macro F1 with mean and standard deviation. Handles edge cases with small labeled sets.

New Enhanced Evaluation Metrics

  • MAP@K (Mean Average Precision at K)
  • nDCG@K (Normalized Discounted Cumulative Gain)
  • ARI (Adjusted Rand Index)
  • Silhouette Score

New Hyperparameter Tuning

Grid search and random search with automatic evaluation. Accepts custom parameter grids and scoring functions. Returns best parameters, score, and full results history.

New CLI Tool

Full command-line interface: pycleora embed|info|benchmark|similar. Supports all algorithms, output formats (npz/txt), and dataset operations. Entry points configured for pip install.

New Benchmarking Suite

Automated comparison of 9 algorithms across 6+ datasets. Measures accuracy, macro F1, time, memory, silhouette score, and modularity. Produces formatted comparison tables.

New Graph Generators

5 graph models: Erdos-Renyi, Barabasi-Albert, Stochastic Block Model, Planted Partition, Watts-Strogatz. Returns edge lists compatible with SparseMatrix.

New Edge Feature Embedding

Multi-dimensional features per edge. Three combination modes: concat, mean, edge_only. Integrates edge attributes into the embedding process.

Improved pip install

Both setup.py and pyproject.toml configured for maturin builds. CLI entry points auto-installed. Version synced to 3.0.0.

Improved Test Suite

81 tests (up from 54) covering all new modules. Full suite runs in under 4 seconds.

Fixed Cross-validate crash with small labeled sets

Now gracefully handles graphs with fewer labeled nodes than requested folds.

Fixed Barabasi-Albert crash for small graphs

Fixed edge case where m >= num_nodes caused errors.

Fixed nDCG edge alignment

Fixed filtering logic to align with MAP@K for consistent evaluation.

v2.1.0

January 2026

New Heterogeneous Graph Support

Multi-type node and edge support. Per-relation embedding, metapath-based embedding, and homogeneous graph export.

New Attention-Weighted Embedding

Embedding with learned attention weights using temperature-based softmax normalization.

Improved Visualization Module

Community-colored t-SNE plots. Custom colormaps. Save to PNG/SVG.

v2.0.0

October 2025

Breaking New Algorithm API

All alternative algorithms moved to pycleora.algorithms module. Unified parameter naming.

New DeepWalk and Node2Vec

Random walk-based embeddings with configurable walk length, number of walks, and BFS/DFS bias parameters.

New GraRep, HOPE, NetMF, ProNE, RandNE

5 spectral and matrix factorization algorithms added. Each optimized for different graph properties.

New Node Classification

Built-in nearest centroid, KNN, MLP classifiers. Accuracy, macro/weighted F1 scoring.

New Link Prediction Metrics

AUC, MRR, Hits@K scoring for link prediction tasks.

Improved Louvain Community Detection

Binary adjacency matrix fixes. Proper self-loop removal. Correct modularity computation.

v1.2.0

August 2025

New Multi-Scale Embedding

Concatenation of embeddings at different propagation scales. Captures local and global structure.

New Node Feature Integration

Weighted combination of structural and attribute-based embeddings.

Improved I/O Module

NPZ save/load, NetworkX/PyG/DGL export support.

v1.1.0

June 2025

New Dynamic Graph Updates

Add and remove edges from existing graphs. Streaming batch embedding.

Improved Normalization Options

L1, L2, spectral, and no-normalization modes.

v1.0.0

March 2025 — Initial Release

New Core Cleora Implementation

Rust-powered sparse matrix operations with PyO3 bindings. Markov propagation-based graph embedding. Entity embedding with configurable dimensions and iterations.

New Similarity Search

Cosine similarity-based nearest neighbor search. Top-K entity retrieval.

New Column Definition System

Flexible entity type configuration. Complex, transient, and reflexive modifiers.
